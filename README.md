# 中文結構查詢語言
結構查詢語言（Structured Query Language, SQL）始於西元一九七四年之美國，能多而用簡，人盡可用。1980年10月，美國以爲標準，萬國咸用。
結構查詢語言有三操焉：一曰定，二曰纂，三曰運。其基關係代數，其致明所爲而不明所以爲；而今者基英不基漢，不便國人，多語達一意，吾爲之痛恨嘆惋。故作此于結合網。吾亦聞之，阿里棄存儲過程、觸發器，不便調試、耦合過度故也。

## 文件約定
半形爲文法符號，全形爲代碼文字；文法推導依`<尖括號>:表達式`。
- `[方括號]`：可選。
- `(圓括號)`：組合。
- `竪|綫`：多選一。
- `{a,b}`：前一文`a`至`b`遍。
- `{a}`：前一文`a`遍。
- `{a,}`：前一文`a`遍以上。
- `{大括號内漢字}`：任選一字。
- `[...]`：前詞元一遍以上。
- `[、...]`：前詞元一遍以上，以中文逗號（、）分隔，[括號内可接受尾端逗號](doc/syntax.md)。
- `(?=文)`：後以`文`。
- `(?!文)`：不後以`文`。
- `(?<=文)`：先以`文`。
- `(?<!文)`：不先以`文`。
