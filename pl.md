# 操作指令
`<指令>`一以句點終。換行不必書句點。
`<表達式>`見[此](expressions.md)。

## 定名
```
<定名指令>:令{「名」〈品類〉置<表達式>}[、...]。
<賦值指令>:令{「名」置[二元運算符]<表達式>}[、...]。
```
品類不可變。

## 選擇
```
<選擇指令>:若<條件>則。<指令>[否若<條件>則。<指令>...][否則。<指令>]也。|
  若<式>爲<式>則。<指令>[爲<式>則。<指令>...][否則。<指令>]也。
```

## 循環
```
<循環指令>:凡{<條件>|<表達式>中「變量」[、...]皆|<數>遍}。<指令>也。
```

### 跳轉
- `止[<數字>]`：所在第`<數字>`循環終止，自内而外，`<數字>`無則取一。
- `次[<數字>]`：所在第`<數字>`循環迭代一次，自内而外，`<數字>`無則取一。
- `得<式>`：終其術，出一元。

|彼云|我作
|-|-
|`若三大於二者。乃...也。`|`若三大於二則。...也。`
|`若三不大於五者。乃...。若非。乃...也。`|`若三不大於五則。...否則。...也。`
|`爲是百遍。...云云。`|`凡一百遍。...也。`
|`恆爲是。...云云。`|`凡一。...也。`
|`凡「天地」中之「人」。... 云云。`|`凡「天地」中「人」。...云。`

## 調試
`書（式）。`於終端。
|彼云|我作
|-|-
|`吾有一數。曰五。書之。`|`書五。`

## 注疏
`＃注疏`解於人耳目，編譯運行則去之。
|彼云|我作
|-|-
|`批曰。「「文氣淋灕。字句切實」」。`|`＃文氣淋灕。字句切實。`
|`注曰。「「文言備矣」」。`|`／＊文言備矣＊／`
|`疏曰。「「居第一之位故稱初。以其陽爻故稱九」」。`|`／＊居第一之位故稱初。以其陽爻故稱九＊／`
