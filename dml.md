# 數據操控
[PostgreSQL](https://www.postgresql.org/docs/current/dml.html)
## 1 增（INSERT）
```
增「表名」列（「列名」、...）置{<式組>、...}[重則更<列置>]。
<式組>:（<式>、...）
<列置>:「列」置{<式>|初}[、...]
```
`<式組>`：式數視列數，不等者[報錯](./errors.md)「入數不等：*列數*列，得*式數*」。

## 刪 (DELETE)
```
刪「表名」若<條件>[限<數>行]。
```

## 更（UPDATE）
```
更「表名」之{「列」置{<式>|初}[、...]}若<條件>
  [序以{「列名」|<式>、...}{升|降}][限（數）行]。
```

## 取 (SELECT)
```
<取語句>:取{式[名「新列名」][、式[名「新列名」]...]|各列}於「表名」[若<條件>]
  [組以{「列名」|<式>、...}[使<條件>]]
  [序以{「列名」|<式>、...}{升|降}]
  [限（數）行][交|并|差<取語句>]。
```

## 表（TABLE）
```
表「表名」。
```
若`取各列於「表名」。`

## 子查詢
子查詢，`<取語句>`外添括號，若作唯一參數者不必添。
