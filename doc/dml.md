# 數據操控
[PostgreSQL](https://www.postgresql.org/docs/current/dml.html)
## 1 增（INSERT）
```
增「表名」列（「列名」、...）置(<式組>、...)[重則更<列置>][返<式>名「參數名」]。
<式組>:（<式>、...）
<列置>:「列」置(<式>|初)[、...]
```
`<式組>`：[`<式>`](./expressions.md)數視列數，不等者[報錯](./errors.md)「入數不等：*列數*列，得*式數*」。

## 2 刪 (DELETE)
```
刪「表名」若<條件>[限<數>行][返<式>名「參數名」]。
```

## 3 更（UPDATE）
```
更「表名」之(「列」置(<式>|初)[、...])若<條件>
  [序以(「列名」|<式>、...)(升|降)][限（數）行][返<式>名「參數名」]。
```

## 4 取 (SELECT)
```
取<取式>。
<取式>:(式[名「新列名」][、式[名「新列名」]...]|各列)於「表名」[若<條件>]
  [組以(「列名」|<式>、...)[使<條件>]]
  [序以(「列名」|<式>、...)(升|降)]
  [限<數>行][交|并|差取<取式>]。
```

## 5 表（TABLE）
```
<表語句>:表「表名」。
```
若`取各列於「表名」。`

## 6 子查詢
子查詢，「取」語句外添括號，若作一元函數參數者不必添。

## 7 調（CALL）
用術之法：`調「術名」[（式、...）]`。
