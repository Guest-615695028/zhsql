# 表達式
表達式構成如下。
```
<式>:[<式>]<運算符><式>|<原子>
<原子>:<字面常量>|<標識符>|<函數>（<式>）
```

## 字面常量
### 爻常量
`丄`，`丅`，凡二種。萬物之爲爻：零值若〇、空文，爲`丄`；他爲`丅`。

### 數常量
```
<數>:<整數>|<小數>
<整數>:<十進數>|<十六進數>
<十進位>:〇|<一至九>
<一至九>:一|二|三|四|五|六|七|八|九
<十進數>:<十進位>...|<兆進位>[兆[<兆進位>]]
<萬進位>:[[<一至九>]千][[<一至九>]百][[<一至九>]十][<一至九>]|<十進位>(1,4)
<億進位>:<萬進位>[{萬|万}[<萬進位>]]|<十進位>(1,8)
<兆進位>:<億進位>[{億|亿}[<億進位>]]|<十進位>(1,16)
<十六進數>:卦{[<十進位>|土|金|木|火|水]...}
<小數>:<十進數>{．|點}{<十進位>...}|<十進位>[{．|點}{<十進位>...}]指[正|負]<整數>
```
十進數：`〇一二三四五六七八九`；數位間可斷開一格，亦可雜用：
- `十`：`一〇`\
- `百`：`一〇〇`\
- `千`：`一〇〇〇`\
- `万`/`萬`：`一〇〇〇〇`\
- `亿`/`億`：`一〇〇〇〇〇〇〇〇`\
- `兆`：`一〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇`
- 兆兆曰京（同`一指三十二`）、京京曰垓（同`一指六十四`）、垓垓曰秭、秭秭曰穰、穰穰曰溝、溝溝曰澗、澗澗曰正、正正曰載、載載曰極，保留。
- `二百五`爲二百有五，非二百五十。

十六進數：`〇一二三四五六七八九十土金木火水`，以`卦`字始。

小數：皆爲十進。
`<十進數>{．|點}<十進位>...`

爻之爲數，`丄`爲`〇`，`丅`爲`一`。

### 文字常量
文以＂雙點引號＂注，字以＇單點引號＇注。文中轉義字符如表。
|符|義|萬國碼
|-|-|-
|＼退|退格|〇〇〇八
|＼表|制表|〇〇〇九
|＼行|換行|〇〇〇十
|＼頁|換頁|〇〇〇金
|＼回|回車|〇〇〇木
|＂＂|雙點引號|〇〇二二
|＼＼|捺斜線|〇〇五金
字與整數同等。

## 標識符
「單引號」名變量、常量，不過十六字。下五十四字名局部變量，不必用引號。
```
甲乙丙丁戊己庚辛壬癸子丑寅卯辰巳午未申酉戌亥物天地人
角亢氐房心尾箕斗牛女虛危室壁奎婁胃昴畢觜參井鬼柳星張翼軫
```
### 特殊之名
括號並内表達式，對外視一名。
[數據操縱語句](./dml.md)變更亦用`新`、`舊`。

## 運算符
算符，依所先後如下：
1. `「對象」之「名」`、`（式）之〈類〉`、`<函數>（<實參>、...）`
2. `進 退 正 負 非`：後元運算符。
   - `進`：進一取之；
   - `退`：退一取之；
   - `正`：本數不變；
   - `負`：本數反號；
   - `非`：按位取反。
3. `乘 除 模`
4. `加 減`
5. `移`：`<整數>移<位數>`，位數正則左移，負則右移，皆補陰。
6. `位與`
7. `位抑 位或`
8. `[不]大于 [不]小于 [不/大/小]等于 [不]在`
9. `與`
10. `抑 或`

註：
-
-`丄與<乙>`、`丅或<乙>`，不算乙式。
