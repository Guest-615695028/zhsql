# 數據定義
[PostgreSQL](https://www.postgresql.org/docs/current/ddl.html)
建、去、改、清，凡四類。
有而欲復建，**若無**（IF NOT EXISTS）二字明則不作，不明則覆之。


## 1 建（CREATE）

### 1.1 模式（SCHEMA）
數據庫下設模式，模式下設對象（表、視圖、過程等）。
```
建模「模式名」［若無］。
```

### 1.2 表（TABLE）
表有行（音航）序，自一而增。列名不用「行」字。
```
建表「表名」[列]（「列名」<列義>[、...]）[繼「表名」][限「限制名」置（式）]。
建表「表名」如「表名」。
建表「表名」取<取式>。
<列義>:[〈品類〉[置（式）][獨]|依「表名」]。
```

- 彼以限（CONSTRAINT）字明主（PRIMARY）、獨（UNIQUE）、依（FOREIGN KEY），徒增文長，吾不用也。
- `〈品類〉`視[此](../guide.md)。
- 置：增行不明此列則置其式之值，其式不可見其列，不之知之故也。
- 獨：不得重複。
- 索：索引。
- 依：所謂「外鍵」者是，不逾庫。所依刪則歸零，所依更則更之。
- `<取式>`：[「取」語句](./dml.md)去首「取」字，下同。

### 1.3 視圖（VIEW）
視圖引表以作，可[`取`](./dml.md)，不可`增刪更`。
```
建引「視圖名」取<取式>。
```

### 1.4 函數（FUNCTION）
函數，算式相連。
```
建法「函數名」入（「參名」〈品類〉，...）{置<表達式>|得〈品類〉<計算語句>}。
```
`<計算語句>`視[統一文言編程語言](../README.md)。

### 1.5 存儲過程（PROCEDURE）
過程，高頻重複者可組合，以便民用。
```
建術「過程名」入（「參名」〈品類〉，...）｛｝
```
按：操作語句中有「取」「告」等語句皆輸出。

### 1.6 觸發過程（TRIGGER）
觸發器，隨[`增刪更`](./dml.md)而作。
```
建術「過程名」[先|後]「表名」[增][刪][更]<起止>。
<操作語句>
術成。
```

### 1.7 用戶（USER）
表以行索。行外索引，千表不過一，萬表不過三。

## 2 去（DROP）
無而去之，[告](./errors.md)`無是模「對象名」`、`無是表「表名」`等。
```
去模「模型名」。
去表「表名」。
去法「函數名」。
去術「過程名」。
去引「視圖名」。
```
去模，則諸事物盡去。

## 3 改（ALTER）
無而改之，亦告`無是模「對象名」`、`無是表「表名」`等。

### 3.1 模型（SCHEMA）
```
改模「模型名」名「新模型名」。
```

### 3.2 表（TABLE）
```
改表「表名」[<方法>、...]。
<方法>:
  |增列（「列名」<列義>、...）
  |列「列名」[名「新列名」][〈品類〉][置（式）][獨|去獨]
  |列「列名」[依「表名」]。
  |去列「列名」|（「列名」[、...]）
  |增限「限制名」（式）
  |去限「限制名」
  |<通用改法>
<列義>:〈品類〉[置（式）][獨]|依「表名」。
<通用改法>:名「新名」|遷「模型名」|與「用戶名」
```

表文不動。

### 3.3 其他
```
改引「視圖名」<通用改法>。
改法「函數名」<通用改法>。
改術「過程名」<通用改法>。
```

## 4 清表
```
清[表]「表名」。
```

表文盡去。
