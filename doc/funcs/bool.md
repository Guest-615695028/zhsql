# 爻函數
## 爻之運算符
一元運算符：`非`在前，`真 假 非真 非假`在後。
|元|丄|〇|丅
|-|-|-|-
|非|丅|〇|丄
|真|丄|丄|丅
|假|丅|丄|丄
|非真|丅|丅|丄
|非假|丄|丅|丅

二元運算符：`與、抑、或`，後元`丄〇丅`依次
|前元|丄|〇|丅
|-|-|-|-
|與|丄丄丄|丄〇〇|丄〇丅
|抑|丅〇丄|〇〇〇|丄〇丅
|或|丄〇丅|〇〇丅|丅丅丅

- `丄與<乙> 〇抑<乙> 丅或<乙>`，不算`<乙>`式，不用`<乙>`中術。

## 比較運算符
- `大于` `小于` `等于`
- `不大于` `不小于` `不等于`
- `大等于` `小等于`
- `在` `不在`

## 轉換
數之爲爻：正數爲`丅`，〇爲`〇`，負數爲`丄`。
餘者不可直轉。
