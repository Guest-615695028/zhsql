# 三進制計算機應用
西曆二〇二三年九月十八日，華爲出專利云「三進制邏輯門電路、計算電路、芯片以及電子設備」（專利號：CN119652311A）。是乃中國人之壯舉，[必以破英文代碼壟斷](https://zh.wikipedia.org/wiki/非基於英語的編程語言)，故吾必善[之](https://zh.wikipedia.org/wiki/中文編程語言)。

不用`卦`；`爻`改爲`丄`、`〇`、`丅`三種取值。

## 八十一進制字面量
八十一進制字面量數字如下，以`玄`字始。
```
中周礥閑少戾上干𤕠
羨差童增銳達交䎡傒
從進釋格夷樂爭務事
更斷毅裝眾密親斂彊
睟盛居法應迎遇竈大
廓文禮逃唐常度永昆
減唫守翕聚積飾疑視
沈内去晦瞢窮割止堅
成䦯失劇馴將難勤養
```
`礥`（U+07925，基本區）：簡體在擴展庚區（U+309C7）。
`𤕠`（U+24560，擴展乙區）：從左「爻」右「守」，爲「疏」古字。
`䦯`（U+049AF，擴展甲區）：簡體在擴展丙區（U+2B535）。
